# AVP_proje
Uzay Simülasyonu (C dilinde)
Proje Hakkında
Bu proje, C dilinde geliştirilmiş konsol tabanlı bir uzay simülasyonu uygulamasıdır.
Kullanıcı, bir bilim insanını yöneterek farklı uzay koşullarında deneyler yapabilir ve sonuçları görebilir.
Projenin amacı, farklı gezegenlerde fiziksel kuralların nasıl değiştiğini simüle etmek ve kullanıcıya bilimsel bir deney ortamı sunmaktır.

Kullanım
1-Programı başlatın.
2-Bilim insanının ismini girin.
3-Menüden yapmak istediğiniz deneyi seçin.
4-İstenen bilgileri girin (hız, kütle, hacim vb.).
5-Program tüm gezegenler için sonuçları hesaplayıp ekrana yazdıracaktır.
